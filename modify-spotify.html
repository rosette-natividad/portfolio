Modify Spotify

Problem Solving Through Design

☆ ★ ☆ ★ ☆

Modify Spotify was a group of six members, including myself, for our Problem Solving Through Design project in my Social Informatics course in Fall 2022.
Image from Spotify.
Goal

Spotify is a popular music streaming platform. Our goal was to improve Spotify’s user experience through design, whether it be remodeling existing features or designing new features. There’s always room for improvement. You can read more in depth about the Problem Solving Through Design project here.

Methods

We indicated some stress cases through conducting think-aloud interviews and research.

Think-Aloud Interviews

We interviewed a range of participants: first-time, moderate, and frequent users of Spotify. With our participants we asked them to complete tasks and speak aloud about their experiences (ex: if the task is easy or difficult, etc.). Below is a list of those tasks:

  • Playlists: creating a playlist, adding and removing songs, rearranging the playlist
  • Song Recommendations: exploring personalized playlists through your home screen, use hide and not interested buttons, shuffle function
  • Friend Activity: find and add friends, see what friends are listening to, make collaborative playlists
  • Search Bar: search for specific songs/artists, create playlists from search results, use keywords or genres to find songs
  • Sharing and Queuing Songs: sharing songs with friends or social media, reorder songs in a queue
  • Downloading Songs: play songs offline, manage downloaded songs by redoing the function
  • Bluetooth: connecting/disconnecting to other devices, start a group listening session
  • Spotify Premium: subscribing and unsubscribing, change plans
Stress Cases
  • Shuffle Feature: Does not truly shuffle a playlist, it is a curated shuffle based on the data it collects about the user and lowers certain songs based on its priority; caused by another feature, Automix
  • Liked Songs Organization: Acts as a Saved Songs organizational tool, and users struggle to have more organization within their profile because the heart-shaped icon is now replaced with a plus sign
  • Downloading Songs: Users cannot download single songs, only playlists. However, a first time user of Spotify struggled to download songs because the button to complete this function wasn’t clear
  • Spotify Premium: Users have to go off the app and onto the web to subscribe/unsubscribe but they can be charged without even knowing their subscription ended
  • Friend Activity: Spotify struggles to socially engage users and they have no options to see their friends’ top songs and artists because friend activity is only accessible on the desktop

Solutions & Recommendations
Curated Shuffle Feature
  • New name and description for the existing feature Automix.
  • Easier to comprehend how it functions and what it affects
Social Features
  • New Profile button to home page UI: Enhances profile customizations
  • New Friend Activity button to navigation bar: Enables friend activity display on mobile
  • Enhances social interactions within the mobile app
Downloading Music
  • New Download button to Music Player UI: Allows user to download and un-download the song currently playing.
  • New Select Song, Download Song buttons and moved Remove Song button in Edit Playlist UI:
    • Improves the interface and function
    • Allows users to be able to select multiple songs and either remove them from the playlist or download them
  • Great way to save storage, especially for larger playlists
Spotify Mail
  • Added a new Mail button to home page UI: Brings up a new interface to the user that displays important information regarding their account.
    • Premium Renewal/Cancellation
    • New premium sales/offers/trials
    • Beta testing opportunities
    • & More
  • Mail feature added to Spotify’s notification system.
